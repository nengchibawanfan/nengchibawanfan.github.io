搭建自己的翻墙服务器

发表于 2019-02-02

薅亚马逊羊毛搭建十块一年的翻墙服务器

亚马逊搭建翻墙服务器一年只需十块钱！只要十块钱！十块钱，你买不了吃亏！十块钱，你买不了上当！如果没撸过，那么各位看官里面请！！！
为了开阔视野、学习新技术，翻墙去见见大（某）千（某）世（hu）界（b）是很有必要的啊
虽然 翻墙软件不断被封，但是我们学习的热情是必不能减的啊！本文就教大家搭建自己的翻墙服务器
免费云服务也是有限制的，每月750小时，每月100w次请求，15G 好像是这么个限制

如何科学上网

第一步，注册

  1. 首先呢，你要搞一张外国的信用卡，后面会用到。淘宝有专门搞这个的，十块一张，不保证激活，不过我激活三次成功了两次，失败的一次是因为注册亚马逊的时候信息填的有点问题。。。
  2. 然后呢就是注册亚马逊账号了，搜索AWS（Amazon Web Services），进入网址 https://amazonaws-china.com/cn/free/
  3. 注册亚马逊账号，填写个人信息，绑定上面的信用卡。我记得我都是搞一个不用的邮箱，然后信息乱填的
  4. 然后屏幕上会有一个验证码，亚马逊机器人小助手打电话过来，输入屏幕上的四个数字，之后输入 * ，就可以成功激活了

第二步，创建实例

  1. 注册好了以后呢，登录aws，点击右上角我的账户
  2. 右上角用户名边上选择地区，根据自己的需求创建主机所在的位位置，如果只是翻墙，可以比较近的比如东京
  3. 点击启动实例，选择一个Linux系统，推荐使用Ubuntu Server 16.04 LTS(HVM)，记得选择的时候选带有符合条件的免费套餐，选择的时候是会提醒你是不是免费套餐的，所以 不用担心选错的～
  4. 之后点击启动，会让你创建密钥，输入一个名字，之后下载存储秘钥
  5. 启动实例，就可以再你的服务器中看到了，启动过程呢。。。一会就启动好了说明，成功了百分之九十，一直在启动呢，八成是激活失败了！服务器名，dns，ip等，记录下来ip
  6. 只能创建一个免费的哦，你如果创建了多个免费的，那么，你可能就欠费了，你的帐号就被封停了

第三步，连接实例

  1. 点击启动实例后面的连接，里面会有多种连接方式，推荐大家使用ssh连接
  2. 使用ssh连接需要使用先前生成的秘钥
  3. 窗口出现ubuntu@x-x-x表示成功

第四步，安装软件

  1. 提升权限，安装软件
  2. 更新软件源，sudo apt update
  3. 安装python包管理工具 ，sudo apt install python-pip，
  4. 安装科学上网软件，pip install shadowsocks
  5. 安装锐速(锐速不支持Openvz！)TCP加速(可选)
    • wget -N --no-check-certificate https://github.com/91yun/serverspeeder/raw/master/serverspeeder.sh && bash serverspeeder.sh

第五步，配置软件

  1. 编写一个配置文件，在/etc/shadowsocks.json中编写配置（如果没有就创建）

    其实单个用户的配置足够，一个账户也可以多个设备登录

第六步，启动服务器

  1. 在终端中启动服务器ssserver -c /etc/shadowsocks.json -d start
  2. 加入开机自启，编辑/etc/rc.local ，将启动指令粘贴到此
  3. 去服务器配置中打开端口，到自己的实例上，到最右边找到安全组，进入安全组设置，点击左下角的入站，编辑，点击添加规则，在端口范围里填上我们设置的端口，来源下拉框中选择任何位置。
